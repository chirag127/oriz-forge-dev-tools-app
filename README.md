# Oriz Forge — Developer Tools

Oriz Forge is a developer toolbox forge — a growing collection of small, fast utilities for building, smithing, and shipping code. Part of the `chirag127/oriz*` family.

## Stack

- Astro 6
- React 19 (islands)
- Tailwind v4
- `@chirag127/astro-shell` + `astro-chrome` (workspace packages)

## Brand

Product brand: **Forge** — developer toolbox forge, building and smithing code.
